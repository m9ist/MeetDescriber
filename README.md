# for_meets

Сервис обработки аудиозаписей встреч и переговоров.

## Назначение

Автоматизирует превращение аудиозаписи встречи в структурированные документы:

1. **Расшифровка** — подробная транскрипция аудио с сохранением всего сказанного
2. **Смысловая выжимка** — второй проход по расшифровке: извлечение ключевых тезисов, решений и договорённостей
3. **Фоллоу-ап** — структурированный документ с итогами встречи, задачами и следующими шагами

## Статус

Проект в стадии проектирования.
